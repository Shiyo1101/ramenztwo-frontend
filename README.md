# ramenztwo-frontend

PRTimes Hackathon Summer

## 必要な事前準備

以下のツールを事前にインストールしてください：

1. **Node.js**

   - 推奨バージョン: 最新の LTS 版
   - [Node.js 公式サイト](https://nodejs.org/)

## 環境構築手順

### 1. リポジトリのクローン

```bash
git clone https://github.com/Shiyo1101/ramenztwo-frontend.git
cd ramenztwo-frontend
```

### 2. 必要なパッケージのインストール

プロジェクトディレクトリ直下で以下を実行

```bash
npm install
```

### 3. 開発サーバーの起動

```bash
npm run dev
```

## その他コマンド

### 1. リント(Linter)

```bash
npm run lint
```

### 2. フォーマット

```bash
npm run format
```
